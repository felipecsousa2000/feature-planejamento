# Agent Code API

API RESTful de exemplo com Node.js, Express e PostgreSQL, estruturada para desenvolvimento profissional.

---

## 💡 Funcionalidades

- CRUD completo de usuários (`name`, `email`)
- Validação de dados com Zod
- Banco de dados via Prisma ORM
- Estrutura organizada (controllers, services, repositories, routes)
- Testes básicos com Jest
- Configuração pronta para Docker

---

## 🛠️ Stack

- **Runtime:** Node.js 20+
- **Framework:** Express
- **Banco:** PostgreSQL (via Prisma)
- **ORM:** Prisma
- **Testes:** Jest + Supertest
- **Validação:** Zod
- **Docker:** docker-compose disponível

# Cortana Copiloto - Modo AGENT CODE

Um **Co-piloto técnico de desenvolvimento** em Node.js, estilo Cortana, que transforma prompts em código real, gera projetos, arquivos, testes e setups automaticamente.

---

## 💡 Funcionalidades

- Criar projetos Node.js estruturados (Express + ESM)
- Gerar arquivos de código automaticamente
- Estrutura modular: `controllers`, `services`, `routes`
- Validação de inputs pronta para uso
- Testes automatizados com Jest
- Integração opcional com Prisma e PostgreSQL/SQLite

---

## 🛠 Stack

- Node.js 20+
- Express
- Jest
- Prisma (opcional)
- Inquirer (CLI interativa)
- Chalk (logs coloridos)
