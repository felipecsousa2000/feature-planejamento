# Agent Code API

API RESTful de exemplo com Node.js, Express e PostgreSQL, estruturada para desenvolvimento profissional.

---

## 💡 Funcionalidades

- CRUD completo de usuários (`name`, `email`)
- Validação de dados com Zod
- Banco de dados via Prisma ORM
- Estrutura organizada (controllers, services, repositories, routes)
- Testes básicos com Jest
- Configuração pronta para Docker

---

## 🛠️ Stack

- **Runtime:** Node.js 20+
- **Framework:** Express
- **Banco:** PostgreSQL (via Prisma)
- **ORM:** Prisma
- **Testes:** Jest + Supertest
- **Validação:** Zod
- **Docker:** docker-compose disponível
